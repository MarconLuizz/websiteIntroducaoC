Bem-vindo ao Curso de Linguagem C

Aprenda a programar do zero com um curso completo e gratuito.

Conteúdo do Curso

Módulo 1: Introdução e fundamentos de programação

Conheça os fundamentos da linguagem C, sua história e estrutura básica.

Módulo 2: Estrutura de um programa

Aprenda como funciona a estruturação de um programa.

Módulo 3: Variáveis e Operadores

Aprenda sobre tipos de dados, operadores e declaração de variáveis.

Módulo 4: Entrada e saída de dados

Aprenda como fazer a entrada de dados para um programa e como realizar a saída dos dados.

Módulo 5: Estruturas condicionais

Aprenda como criar estruturas condicionais e suas utilidades.

Módulo 6: Estruturas de repetição

Aprenda sobre as estruturas de repetições e laços.

Módulo 7: Funções

Aprenda a criar funções e suas utilizações.

Sobre o Curso

O curso de Linguagem C foi desenvolvido para iniciantes, com aulas teóricas e práticas, usando exemplos e desafios reais.

Inclui vídeos, slides, exemplos e exercícios práticos.
