REA: Fundamentos de Programação em C

Aprenda os conceitos básicos de programação estruturada em C de forma interativa e didática.

Objetivo Geral

Criar uma plataforma interativa com documentações de C, aprendizagem sequencial e vídeo aulas por módulos, de maneira que auxilie iniciantes a entenderem os fundamentos da programação em C.

Objetivos Específicos

  1. Apresentar variáveis, tipos de dados, operadores e estruturas de controle.
  2. Exemplos de código com realce de sintaxe e passo a passo.
  3. Quizzes para verificar compreensão.
  4. HTML semântico, contraste adequado e responsividade.
  5. Divulgação em repositório aberto.
  6. Vídeo Aulas gravadas para cada módulo do curso.
